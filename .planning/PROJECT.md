# HUMAN.AI — Фаза 0: Инфраструктура и онтология

## Что это такое

Backend-ядро Talent Intelligence Platform — фаза подготовки (Phase 0): поднять локальный стек (Neo4j, Qdrant, Redis, FastAPI), закодировать онтологию графа как Pydantic-модели и Cypher-миграции, загрузить тестовых кандидатов вручную, настроить eval-харнес.

Это фундамент для всех последующих фаз (парсер, LLM-экстрактор, retrieval). Без проверенной онтологии и рабочего стека остальные фазы бессмысленны.

## Ключевая ценность

Локальный стек поднимается одной командой, в Neo4j лежат тестовые кандидаты, базовые Cypher-запросы работают — каждый следующий компонент есть куда писать.

## Контекст

Стек выбран и зафиксирован в core_architecture.md: Python 3.11+, FastAPI, Neo4j 5.x, Qdrant, Celery + Redis, BGE-M3 (локально, CPU). Docker Compose поднимает все инфра-сервисы. LLM-провайдер не выбран — решение откладывается до замеров Г1 и Г4 (Фаза 1–2).

Каталоги `api/` и `core/` существуют, но Python-кода внутри нет. Тестов нет.

Целевой пилот: 1 клиент, ~300 резюме, 1 рекрутер, SaaS на нашей инфре.

## Ограничения

- **Стек**: Python 3.11+, FastAPI, Neo4j 5.x, Qdrant, Celery + Redis — изменение требует пересмотра архитектуры
- **LLM**: провайдер не выбран до Фазы 1–2 — не закладывать зависимость в Phase 0
- **Бюджет**: инфра-расходы минимизированы — Docker Compose на одной VM, без облачных управляемых сервисов
- **Онтология**: расширение — по факту боли, не раньше; факт-узел обязателен (провенанс из коробки)

## Ключевые решения

| Решение | Обоснование | Результат |
|---------|-------------|-----------|
| Граф — источник правды, Qdrant — индекс | Vектор находит, граф объясняет; KAG-паттерн | — Ожидается |
| Факт как отдельный узел | Провенанс, версионность, разрешение конфликтов из коробки | — Ожидается |
| BGE-M3 локально на CPU | Мультиязычность, ru-качество, нет зависимости от внешнего API | — Ожидается |
| Celery + Redis (не Airflow) | Минимальный стек, расширяется до Airflow позже | — Ожидается |
| Никаких строк вместо узлов | `Skill: "Python"` — это узел, иначе нормализация невозможна | — Ожидается |
