    ```python
    @staticmethod
    def _experience_id(document_id: str, company: str, role: str, from_date: str) -> str:
        return hashlib.sha1(f"{document_id}|{company}|{role}|{from_date}".encode()).hexdigest()
    @staticmethod
    def _education_id(document_id: str, institution: str, from_date: str | None) -> str:
        return hashlib.sha1(f"{document_id}|{institution}|{from_date}".encode()).hexdigest()
    @staticmethod
    def _contact_id(document_id: str, type_: str, value: str) -> str:
        return hashlib.sha1(f"{document_id}|{type_}|{value}".encode()).hexdigest()
    @staticmethod
    def _fact_id(document_id: str, predicate: str, value: str) -> str:
        return hashlib.sha1(f"{document_id}|{predicate}|{value}".encode()).hexdigest()
    ```

    Skill union+dedup (D-04/D-05 — `.strip()` only, NO canonicalization):

    `write()` — async, graceful-degradation guard FIRST, then ONE transaction:
    ```python
    async def write(self, candidate: ExtractedCandidate, document_id: str) -> None:
        candidate_id = document_id  # D-01
        if self._db is None or not self._db.is_connected:
            logger.warning("graph_writer: Neo4j unavailable — candidate graph not persisted id={}", candidate_id)
            return
        async with self._db.session() as session:
            await session.execute_write(self._write_tx, candidate, candidate_id)
        logger.info("graph_writer: candidate graph written id={}", candidate_id)
    ```

    `_write_tx(self, tx, candidate, candidate_id)` is an async function that runs
    all `await tx.run(STATEMENT, **params)` calls in this ORDER (reproduce seed.py
    sequence, data from candidate):
    1. MERGE_CANDIDATE (id=candidate_id, full_name=candidate.full_name, status="active")
    2. For each contact: MERGE_CONTACT (id=_contact_id(...), type, value) then LINK_HAS_CONTACT
    3. For each skill in the unioned set: MERGE_SKILL (name=skill) then LINK_HAS_SKILL
    4. For each experience:
       - exp_id = _experience_id(document_id, exp.company, exp.role, exp.from_date)
       - MERGE_COMPANY (name=exp.company, industry=None)
       - MERGE_ROLE (title=exp.role, seniority=None)
       - MERGE_EXPERIENCE (id=exp_id, from_date, to_date=exp.to_date, is_current=exp.is_current)
       - LINK_HAS_EXPERIENCE, LINK_AT_COMPANY, LINK_AS_ROLE
       - For each s in exp.skills_mentioned (after .strip()): LINK_USED_SKILL (e_id=exp_id, name=s)  # skill node already MERGEd in step 3 via union
    5. For each education: edu_id = _education_id(...); MERGE_EDUCATION; LINK_HAS_EDUCATION
    6. Fact provenance (D-03, D-07):
       - For each UNIQUE skill: fid=_fact_id(document_id,"has_skill",skill); MERGE_FACT(predicate="has_skill", value=skill, confidence=None, model_version=candidate.model_version, is_current=True, extracted_at=now); LINK_HAS_FACT; LINK_EXTRACTED_FROM(d_id=document_id); LINK_SUPPORTS_SKILL(name=skill)
       - For each experience: fid=_fact_id(document_id,"worked_at",exp.company); MERGE_FACT(predicate="worked_at", value=exp.company, confidence=None, model_version=candidate.model_version, is_current=True, extracted_at=now); LINK_HAS_FACT; LINK_EXTRACTED_FROM(d_id=document_id); LINK_SUPPORTS_EXPERIENCE(e_id=exp_id)

    `now = dt.datetime.now(dt.UTC).isoformat()` computed once at top of tx (exactly
    as pdf.py line 175). `confidence=None` — pass Python None (becomes Cypher null),
    NEVER a float literal (D-02). Education/Contact get nodes+edges but NO Fact (D-03).
    USED_SKILL only for skills_mentioned; a top-level-only skill gets HAS_SKILL but
    no USED_SKILL (D-06). One Fact per unique skill, not per HAS_SKILL/USED_SKILL edge (D-07).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ExtractedCandidate → graph | resume-derived strings (skills, company, role, contacts) become MERGE keys/values |
| caller-stamped provenance → Fact | document_id/model_version come from the ExtractedCandidate (already stamped in extractor _validate, T-05-04) — writer must not let resume text overwrite them |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04 | Tampering | GraphWriter param binding | mitigate | Writer imports constant statements from cypher.py and passes data ONLY as bound `$param` kwargs to `tx.run`; never f-strings Cypher. Acceptance criteria reject any Cypher string construction in graph_writer.py. |
| T-06-05 | Spoofing | Fact provenance | mitigate | `Fact.model_version = candidate.model_version` (caller-stamped), `document_id` is the EXTRACTED_FROM target by id; resume text cannot forge either. `confidence=None` enforced (no invented value, D-02). Grep rejects float confidence. |
| T-06-06 | Repudiation | EXTRACTED_FROM | mitigate | Every Fact links to the parser-created Document by id (provenance audit trail per core_architecture.md §5.8). Integration test asserts the triple is reachable. |
| T-06-07 | Denial of Service | Neo4j outage | mitigate | is_connected guard before session(); outage logs a warning and returns — never crashes the writer or its Celery caller (phase 7). Unit test asserts no crash / session never entered. |
</threat_model>
