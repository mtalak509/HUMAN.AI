### Паттерн MERGE для каждого узла (по образцу migrate.py):

```python
await session.run(
    "MERGE (n:Candidate {id: $id}) "
    "ON CREATE SET n.full_name=$full_name, n.status=$status "
    "ON MATCH SET n.full_name=$full_name, n.status=$status",
    id="c-001", full_name="Алексей Соколов", status="active",
)
```

### Паттерн MERGE для связей:

```python
await session.run(
    "MATCH (c:Candidate {id: $c_id}) MATCH (ct:Contact {id: $ct_id}) "
    "MERGE (c)-[:HAS_CONTACT]->(ct)",
    c_id="c-001", ct_id="ct-001",
)
```

### main():

```python
async def main() -> None:
    settings = get_settings()
    db = GraphDB(uri=settings.neo4j_uri, user=settings.neo4j_user, password=settings.neo4j_password)
    try:
        await db.connect_with_retry()
        if not db.is_connected:
            logger.error("Neo4j unavailable — seed aborted")
            sys.exit(1)
        async with db.session() as session:
            await _seed_candidate(session)
        logger.info("Seed complete: candidate c-001 loaded")
    finally:
        await db.close()
```
